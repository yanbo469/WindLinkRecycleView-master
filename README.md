# WindLinkRecycleView
MVP框架下的RecycleView的双列表的城市选择联动与悬停，适合一切二级联动。
----
实现功能：
----
1.点击左边省份，省份背景改变，顶部显示省份悬停，下面显示省份的城市；<br>
2.滑动右边的城市，顶部省份悬停，左边随着省份的改变而改变；<br>
3.所有控件可点击；<br>
4.仿官方mvpdemo的mvp框架；<br>
5.snackbar的使用。<br>
